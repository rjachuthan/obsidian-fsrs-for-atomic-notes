# Development Workflow

## Reloading the Plugin

The plugin needs to be reloaded after every source code change. Two options:

**Option 1: Manual reload**

1. Open **Preferences**
2. Click **Community plugins**
3. Find your plugin under **Installed plugins**
4. Toggle the switch off to disable
5. Toggle the switch on to enable

**Option 2: Hot-Reload plugin**

The [Hot-Reload](https://github.com/pjeby/hot-reload) plugin automatically reloads your plugin whenever the source code changes. See [forum announcement](https://forum.obsidian.md/t/plugin-release-for-developers-hot-reload-the-plugin-s-youre-developing/12185).
